# EC2U PhD Agreements Tool

The EC2U PhD Agreements Tool provides AI-based textual analysis facilities supporting local PhD coordinators and other
relevant stakeholders in the process of drafting cotutelle PhD agreements.

# Support

- open an [issue](https://github.com/ec2u/phds/issues) to report a problem or to suggest a new feature
- start a [discussion](https://github.com/ec2u/phds/discussions) to ask a how-to question or to share an idea
